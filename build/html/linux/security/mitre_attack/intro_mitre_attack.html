MITRE ATT&CK 平台简介¶

MITRE ATT&CK 是一个全球可访问的基于真实世界观察的对手战术和技术知识库。ATT&CK 知识库被用作在私营部门、政府以及网络安全产品和服务社区中开发特定威胁模型和方法的基础。ATT&CK是一个开放系统，可提供任何个人或组织免费使用。

MITRE ATT&CK (medium) 是该公司的系列blog，提供了关于攻防产品的介绍，可以参考。
