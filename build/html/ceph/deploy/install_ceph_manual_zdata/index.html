Ceph 手工部署zdata集群(暂未成功)¶

我在为 Real Atlas 构建 私有云架构 数据存储层(基于Ceph)，采用了自定义Ceph名字 zdata 。虽然 安装 ceph-mon (zdata) 和 安装 ceph-mgr (zdata) 都可以顺利完成，但是在 添加Ceph OSDs (zdata) 遇到了 ceph_volume 不能正常传递集群名称的问题。我暂时没有找到解决方法，但是又有太多其他部署实践需要完成，所以暂时回归采用 手工部署Ceph 部署默认 ceph 作为集群名称。此项自定义安装实践，我准备后续在虚拟化集群中再做实践尝试。

注解

我在部署 Libvirt集成Ceph RBD 实践发现环境变量 CEPH_ARGS ( CEPH_ARGS 环境变量 )可以传递参数，也就是可以在自定义集群名称传递参数。我后续实践将尝试部署自定义集群
