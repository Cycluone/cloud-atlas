Raspbery Pi OS(Raspbian)¶

Raspberry Pi OS是基于Debain开发的开源操作系统，并且针对 树莓派硬件 进行了优化，所以建议在常规树莓派使用场景采用。Raspberry Pi OS提供了超过3万5千个预先编译打包好的易于安装的软件包。

备注

Raspberry Pi OS基于Debian开发，所以继承了Debian社区大量的软件，易于安装维护。从2022年1月开始，Raspberry Pi OS正式发布了64位操作系统，所以现在可以用树莓派官方OS来取代之前我曾经过渡期使用的 树莓派4b运行64位Ubuntu
