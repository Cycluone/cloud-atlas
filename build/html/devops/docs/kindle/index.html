Kindle在，书未老¶

作为技术工作者，有大量的技术文档是使用pdf格式发布的，我们需要不断学习和研读pdf技术文档，所以我非常推荐使用Kindle来阅读电子书。

原因是:

  • 类似智能手机这样多功能设备让我们不自觉沉溺于消费娱乐活动，无法专注于阅读

  • 互联网虽然能让我们轻易获得信息，但是泥沙俱下，获得真实有用的咨询和知识成本越来越高，甚至得不偿失

Kindle以朴素的黑白文字世界让我们能够专注于相当有价值的知识和见解的汲取，同时给予我们文字想象的空间，让我们独处时不孤独惶恐。

注解

2021年伊始，我尝试以 iPhone之禅 的方式来使用必要的电子设备。

注解

Kindle阅读推荐 书伴 网站，提供很多关于Kindle使用技巧以及书籍推荐，特别是提供了完整的 Kindle字典下载 ，方便我们更好地使用Kindle。
