边缘云K3s¶

边缘云采用 Raspberry Pi 这种性能有限的ARM架构，需要考虑低功耗、无风扇、散热条件苛刻的因素。所以，部署全功能的 Kubernetes Atlas 不仅浪费资源，而且也带来了 复杂系统是如何失败的 的可能，所以我选择精简的 K3s - 轻量级Kubernetes 来实现个人工作室的分布式集群。
