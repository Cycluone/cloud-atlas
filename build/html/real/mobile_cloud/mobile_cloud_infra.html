移动云架构¶

硬件和OS¶

采用 Apple ARM架构芯片M1 Pro MacBook Pro 16” ，运行 Asahi Linux 系统:

  • 通过 KVM Atlas 来运行虚拟机，借鉴 私有云架构 部署一个full Kubernetes Atlas

  • 启用域名 cloud-atlas.io 模拟构建 dev.cloud-atlas.io 开发和持续集成环境

模拟集群¶

  • 采用 kind(本地docker模拟k8s集群) 构建本地容器化 Kubernetes Atlas 集群

  • 采用 ARM硬件环境KVM虚拟化 构建本地运行的虚拟化服务器集群，进一步部署 Kubernetes Atlas 和 OpenShift Atlas

    • 探索 ARM嵌套虚拟化扩展(NEVE: Nested Virtualization Extensions for ARM)
