
===========================
SUSE平台Gluster客户端部署
===========================

SUSE系统环境可以使用 ``lsb-release -a`` 命令检查版本::

   LSB Version:    n/a
   Distributor ID: SUSE
   Description:    SUSE Linux Enterprise Server 12 SP3
   Release:        12.3
   Codename:       n/a

在项目上使用的SUSE是作为GlusterFS客户端，访问在CentOS 7上所 :ref:`deploy_gluster6` 。

.. note::

   SUSE也使用rpm包进行软件安装，但是使用了独特的包管理工具 :ref:`zypper`
