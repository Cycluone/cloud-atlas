==================
SUSE部署SMT服务器
==================

在管理服务器上安装Subscription Management Tool(SMT，订阅管理工具) ，然后就可以从SUSE客户中心镜像所有软件仓库。如果公司已经提供了SMT服务器或者可以从管理服务器上访问一个SUSE Manager server，就不需要这步骤。



参考
=====
