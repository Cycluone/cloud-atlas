====================
HP iLO快速起步
====================

Integrated Lights-Out，简称 iLo，是由HP(Hewlett-Packard)公司推出的专有嵌入式服务器管理技术，提供了带外管理功能。HP iLO结合了服务器主板的iLO ASIC(集成电路)以及增强ASIC的firmware。iLO是服务器强大操作的关键，也是启动时方便设置，系统健康监控以及电源和温度控制的平台。
