====================
NFS网络文件系统简介
====================

为解决本地文件系统的限制(磁盘空间有限、无法多客户端共享访问)，网络文件系统(Network File System, NFS)被发明出来:

- 多个客户端可以共享访问服务器上相同的文件
- 非常便于集中管理NFS服务器上的文件(提供一定的安全性)，例如集中备份、冗灾等

NFS最早是Sun Microsystems公司于1984年发明
