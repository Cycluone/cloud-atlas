=====================
Privoxy代理服务简介
=====================

`Privoxy无缓存web代理 <https://www.privoxy.org/>`_ 是一个提供了增强型过滤功能，能够改进私密性、修改web数据以及HTTP头部和控制访问，移除广告等。Privoxy提供了灵活等配置，能够定制适合小型局域网共享上网。
