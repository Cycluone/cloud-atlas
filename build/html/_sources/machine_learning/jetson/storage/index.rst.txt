=================================
Jetson 存储
=================================

和 :ref:`pi_storage` 相似，Jetson Nano作为ARM基础的计算机系统，存储子系统对系统运行性能有很大影响，默认采用缓慢的SD卡将拖慢系统运行。所以，我将尝试改进存储部署，提高Jetson运行效率。
