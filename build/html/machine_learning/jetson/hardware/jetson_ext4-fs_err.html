EXT4文件系统fsck¶

采用 EXT4文件系统fsck 进行修复，实践最终修复是通过将TF卡取下，通过USB转接器，插入到另外一个主机上执行 fsck 修复。修复完成后，上述Ext4文件系统错误不再出现，系统运行稳定。
