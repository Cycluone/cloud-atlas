Termux开发环境¶

在探索 Android 移动工作 ，我采用在 Termux - Andorid终端 中安装必要的软件并配置适合工作的环境

安装软件包¶

  • 安装工具:

termux中安装软件包(apt)¶
 1# sudo工具
 2pkg install tsu
 3# ip工具和vim
 4pkg install iproute2 vim git
 5# openssl工具
 6pkg install openssl-tool
 7# 代理服务
 8pkg install squid
 9# 下载工具
10pkg install axel

环境配置¶

  • 将默认SHELL改成 Termux ZSH

  • ssh密钥 配置

  • ssh多路传输multiplexing加速 配置

开发软件¶

Termux Development Environments 提供了termux软件开发，信息科学教育以及实验的软件列表:

  • 安装 python , c , go , rust :

termux中安装软件包(apt)¶
 1# c/c++
 2pkg add clang
 3# Go
 4pkg add golang
 5# NodeJs
 6pkg add nodejs
 7# Python
 8pkg add python
 9# Rust
10pkg add rust
11# Swift
12pkg add swift

vim¶

采用 我的vim配置 定制，但是需要安装编译依赖:

termux安装编译YCM依赖软件包¶
apt install vim-python clang python cmake
