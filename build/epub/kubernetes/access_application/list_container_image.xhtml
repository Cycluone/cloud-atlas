列出集群的容器镜像

列出指定pod中所有container

列粗一个pod中所有容器:

kubectl get pods [pod-name-here] -n [namespace] -o jsonpath='{.spec.containers[*].name}'

举例:

kubectl --kubeconfig admin.kubeconfig -n kube-system get pods apiserver-56c6db8b58-8l4dn -o jsonpath='{.spec.containers[*].name}'
