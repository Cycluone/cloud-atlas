========================
使用MkDocs撰写文档
========================

文档布局
=========

在MkDocs中，文档就是常规的Markdown文件，存放在 ``docs_dir`` 配置对应的目录(默认是 ``docs`` 目录)::

   mkdocs.yml
   docs/
       index.md
