====================
a-shell命令
====================

vim
======

``a-shell`` 原作者编译原生 :ref:`vim` ，但是 :ref:`nvim` 非常复杂，难以移植。

git替代
=========

a-shell命令没有提供 :ref:`git` ，但是提供了一个原生iOS命令 ``lg2`` 作为 ``git`` 的clone，可以完成相同的工作
