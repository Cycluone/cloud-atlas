===============
Gentoo VSCode
===============

安装准备工作
=============



.. note::

   如果是开源信仰，可以选择 `vscodium <https://vscodium.com/>`_ : VSCode的MIT开源分支，去除了微软内建在VSCode中的跟踪部分信息(反馈给微软改进软件)，并完全使用MIT授权

   不过 vscodium 安装有点麻烦(编译后上传gitlab再下载分发，绕开授权法律限制)，并且微软对于软件跟踪相对节制(相对而言)，普通用户可以直接使用 VSCode。
