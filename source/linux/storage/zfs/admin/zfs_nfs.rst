=============
ZFS NFS
=============

ZFS内置NFS功能为企业级应用提供了方便的共享存储，在很多应用领域有广泛应用:

- :ref:`machine_learning` 大量的训练共享数据
- 医疗影像行业存储大量的医疗图片
- ...

.. note::

   本文实现ZFS NFS服务配置，为后续构建 :ref:`zfs_infra` 的企业应用提供基础
