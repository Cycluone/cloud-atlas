=====================
使用netplan配置网络
=====================

Ubuntu发行版默认使用 `Netplan <https://netplan.io>`_ 配置网络接口，例如我在在 :ref:`pi_4_network` 中就使用了netplan。netplan支持后端使用 ``networkd`` 或者 ``network-manager`` 进行管理配置。

======
