=============================
Docker运行CoreDNS
=============================

除了在Kubernetes部署时通过 ``kubeadm`` 自动完成CoreDNS部署，也可以通过 :ref:`docker` 来运行独立的CoreDNS。这种场景通常是为了解决一些特定场景的独立部署。
