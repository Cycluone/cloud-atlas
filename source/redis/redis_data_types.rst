===================
Redis数据结构类型
===================

Redis支持5种不同数据结构类型:

String字符串
==============

String字符串是最基本Redis值:

- Redis字符串数据可以存储任何类型数据(binary safe)，例如JPEG图形或者序列化Ruby对象
- 字符串类型最大存储512MB长度
